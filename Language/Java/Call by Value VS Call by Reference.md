## Call by Value
- 값에 의한 호출
- 값을 복사해서 새로운 함수로 넘기는 호출 방식
- 원본 값 변경 X

## Call by Reference
- 참조에 의한 호출
- 주소값을 인자로 전달하는 호출 방식
- 원본 값 변경 O
