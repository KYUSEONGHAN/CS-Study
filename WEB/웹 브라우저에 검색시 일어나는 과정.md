## 전체 과정
1. 웹 브라우저에 url 입력하고 엔터키 입력
2. 웹 브라우저가 도메인명의 Ip 주소 조회
3. 웹 브라우저가 서버와의 tcp 연결
4. 웹 브라우저가 http 요청을 서버로 전송
5. 웹 서버가 요청을 처리하고 응답을 다시 전송
6. 웹 브라우저가 콘텐츠 렌더링
