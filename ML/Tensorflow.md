## Tensorflow
- 구글에서 만든 딥러닝을 쉽게 구동할 수 있게 하는 라이브러리.
- 머신러닝을 위한 오픈소스 라이브러리
- 기본적으로 C++로 구현되어 있고, python/java/go 등의 다양한 언어를 지원함
- python이 최우선 언어이며, 대부분의 편의 기능이 python 라이브러리로 구성됨.

## Tensorflow의 특징
- 코드 수정 없이 CPU/GPU 모드로 동작 가능함.
- TensorBoard로 학습과정을 추적 및 시각화 가능
